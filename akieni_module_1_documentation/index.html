Introduction

HTML et CSS sont les langages de base du développement web. HTML permet de structurer la page, tandis que CSS permet de gérer le style et l'apparence.

Cette documentation vous guidera à travers les concepts fondamentaux de ces langages.

Les bases du HTML

HTML (HyperText Markup Language) est le langage utilisé pour structurer les pages web.

Exemple de base :


<!DOCTYPE html>
<html>
    <head>
        <title>Page Web</title>
    </head>
    <body>
        <h1>Hello World</h1>
    </body>
</html>

Quelques balises importantes :

  • <p> pour les paragraphes
  • <a> pour les liens
  • <img> pour les images
  • <ul> et <li> pour les listes
Les bases de CSS

CSS (Cascading Style Sheets) est utilisé pour styliser les pages HTML.

Exemple de style de base :


body {
    background-color: lightgray;
}
h1 {
    color: blue;
}
p {
    font-size: 16px;
}
Les Selecteurs CSS

Les sélecteurs CSS permettent de cibler des éléments HTML.

Exemples de sélecteurs :


h1 {
    color: red;
}
.class-name {
    font-weight: bold;
}
#id-name {
    background-color: yellow;
}
p:hover {
    color: green;
}
Responsive Design

Le responsive design permet aux sites de s'adapter aux différentes tailles d'écran.

Utilisation des media queries :


@media (max-width: 768px) {
    body {
        background-color: lightblue;
    }
}

Outils pour un design responsive :

  • Flexbox
  • Grid
  • Unités relatives (% et rem)
